# Academic Exchange Spain — Premium Launch Website

Static website ready for GitHub Pages.

## Upload instructions
1. Upload all files and folders to the root of the GitHub repository.
2. In GitHub: Settings → Pages → Deploy from a branch → main → /(root).
3. Save and wait 1–3 minutes.
